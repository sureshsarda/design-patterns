* design-patterns
  This repository contains sample code and simulation for some of the famous design patterns.
  The patterns include:
    - Factory Method
    - Builder
    - Composite
    - Prototype
    - Observer
    - Facade
    - Bridge
    - Abstract Factory
    - Proxy
    - Chain of responsibility
    - Visitor
    - Singleton
    - Command
    - State
    - Strategy
    - Adapter
    - Decorator
    - Flyweight
    - Mediator
    - Iterator
    - Momento
    - Template Method
    - Interpreter
